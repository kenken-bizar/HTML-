Ringkasan HTML Apa itu HTML? Singkatan dari HyperText Markup Language → bahasa markah untuk membuat halaman web. Menggunakan tag sebagai penanda untuk menyusun struktur & tampilan konten. Diibaratkan seperti batu bata untuk membangun rumah → fondasi dasar website. HTML & Pelengkapnya CSS → memperindah tampilan web. JavaScript → membuat web interaktif & “hidup” (fungsi, efek, logika). Tahun 1999 → mulai dikembangkan HTML5 oleh W3C. HTML5 = standar modern: dukungan multimedia, form lebih canggih, aplikasi web interaktif, semantik lebih baik, aksesibilitas, dan didukung semua browser modern. lebih ringkass HTML = bahasa markah buat bikin web (fondasi dasar, pakai tag). CSS = styling/tampilan, JavaScript = interaktif/efek. Fungsi = atur struktur halaman, tampilkan konten, bikin link/navigasi. Sejarah = dibuat Tim Berners-Lee (1990-an), berkembang sampai HTML5 (standar modern, dukung multimedia & interaktif). Fungsi Komentar dalam Kode: Dokumentasi → memberi penjelasan pada bagian kode agar mudah dipahami kembali. Debugging → menonaktifkan kode sementara tanpa perlu menghapusnya. Kolaborasi → memudahkan anggota tim lain memahami tujuan dan perubahan kode. Komentar di HTML Definisi → teks di dalam kode yang tidak ditampilkan browser, dipakai untuk catatan/penjelasan. Sintaks → Manfaat → dokumentasi kode, memudahkan pemahaman, bisa nonaktifkan kode sementara tanpa hapus.
